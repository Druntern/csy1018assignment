MGP in UI Design

Menu Bar

Samsung website has a current menu bar which inspired me in my website. It’s clear and simple, with little details that make the different. This is a good example of current UI’s.

Overlay Menu

The way this website did the menu is very modern and minimalist. I liked the idea of covering the whole screen. Even though, I only applied this type of menu for smaller screens like mobile phones and tablets because I wanted to keep my menu bar in desktops as it is just one click and go.

Hero Images

Most current websites nowadays use hero images which are big pictures that cover the screen from one side to another and normally, there are some titles or buttons hovering it. When I go to a website like this, I immediately get a good impression and another good aspect about it, is that we automatically know what kind of page we are.

Many attempts were made before coming with the final product of this website, but I was never satisfied enough so I searched for other websites that had good, modern and current designs. Then I ended up making some sketches in the paper, so it could be easier for me to build my website as I am still a beginner in this specialization. Also, me and some colleagues shared our designs, so we could get some ideas from each other and come up with the best design we can.

I got inspired in the Samsung’s navigation bar because it’s simplicity and modernity is just brilliant and it follows the current design.

A logo was done for this website as well because it gives an extra good looking to it. It gives confidence to the user when the look is clear.

The pattern and colour schemes were kept through the website to keep the coherence in to make it feel like a whole product.

Another key that was a focus, was the responsive layout. It is kept for the whole website for desktop, tablet and mobile phone.
